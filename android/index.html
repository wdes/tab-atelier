tab-atelier Android app (ta-remote)

Signed release APKs of the remote-control app, newest first. Each filename carries the commit short SHA.

Installing

Download the newest .apk on your Android device and open it. Android asks you to allow installing from this source the first time (Settings → Apps → Special access → Install unknown apps).

Self-signed (not on the Play Store). It talks to a tab-atelier headless host over its HTTP API — pair by scanning the QR from the desktop share menu, then set the host’s public URL for remote access.
